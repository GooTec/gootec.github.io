# Time Machine 이미지 파일 안내

이 폴더에는 Time Machine 컴포넌트에서 사용되는 이미지 파일들이 포함되어 있습니다.
현재 파일들은 텍스트 플레이스홀더이며, 실제 사용을 위해서는 적절한 이미지 파일로 교체해야 합니다.

## 필요한 이미지 파일
1. `handong-university.jpg` - 한동대학교 캠퍼스 이미지 (권장 크기: 800x500px)
2. `the-stranger.jpg` - 알베르 카뮈의 '이방인' 책 표지 이미지 (권장 크기: 800x500px)
3. `hwaseong-marathon.jpg` - 화성시 경기 가족 마라톤 관련 이미지 (권장 크기: 800x500px)

## 이미지 교체 방법
1. 적절한 이미지 파일을 준비합니다 (JPG 또는 PNG 형식 권장).
2. 위의 파일명과 동일하게 저장하여 이 폴더에 있는 기존 파일을 덮어씁니다.
3. 프로젝트를 다시 빌드하면 새 이미지가 웹사이트에 적용됩니다.

## 새 이미지 추가하기
새로운 이벤트를 추가할 때 필요한 이미지는 이 폴더에 추가하고, src/pages/TimeMachine.js 파일의 timelineData 배열에서 해당 이미지 경로를 참조하세요.
